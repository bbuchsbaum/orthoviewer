# Run a minimal orthoviewer Shiny example

This launches a small Shiny app that demonstrates the
[`ortho_viewer()`](ortho_viewer.md) widget and the
[`ortho_proxy()`](ortho_proxy.md) API using synthetic background and
overlay volumes.

## Usage

``` r
orthoviewer_example_app()
```

## Value

A Shiny app object, invisibly.
